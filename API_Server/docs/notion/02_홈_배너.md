# GET /home/banner — 히어로 배너

## 기능 개요

홈 화면 상단 히어로 배너 5개를 반환한다. 개인화 추천 우선, 없으면 인기 콘텐츠로 fallback.

## 기본 기능

**요청**:
```
GET /home/banner
Authorization: Bearer {token}
```

**처리 로직**:
1. `serving.hybrid_recommendation`에서 유저 기반 추천 조회 시도
2. 데이터 없으면 `serving.popular_recommendation`에서 인기순 fallback
3. `series_nm` 기준 중복 제거 후 top 5 반환

**응답**:
```json
{
  "items": [
    {
      "series_nm": "기생충",
      "title": "기생충",
      "poster_url": "/posters/parasite.jpg",
      "category": "영화",
      "score": 0.95
    }
  ],
  "total": 5
}
```

## 예외사항

- 추천/인기 데이터 모두 없으면 **빈 배열 반환** (에러 아님)
- hybrid_recommendation 테이블 미생성 시 자동 fallback (Exception 무시)

## 제약사항

- 인증 필수
- hybrid_recommendation 미적재 시 항상 popular fallback
- 시리즈 중복 제거: 동일 series_nm이 여러 rank에 있을 경우 첫 번째만

## 업스트림 의존성

| 테이블 | 컬럼 | 타입 | 용도 |
|--------|------|------|------|
| `serving.hybrid_recommendation` | 전체 | - | 개인화 추천 (primary) |
| `serving.popular_recommendation` | `ct_cl`, `rank`, `vod_id_fk`, `score` | VARCHAR/SMALLINT/REAL | 인기순 (fallback) |
| `public.vod` | `series_nm`, `poster_url`, `ct_cl` | VARCHAR/TEXT | 메타데이터 JOIN |
