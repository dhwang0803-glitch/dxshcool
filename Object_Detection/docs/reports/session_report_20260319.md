## 2. yaml/코드 변경 내역

### clip_queries_ko.yaml (v4)
- 8종 → 2종 (음식 59쿼리 + 관광지 30쿼리)
- 양식/디저트/카페 12쿼리 추가
- 한국 관광지 12쿼리 추가 (벽화골목, 케이블카, 출렁다리 등)
- negative 8개 추가 (실내수영장, 호텔로비, 마트진열 등)
- threshold: 0.30

### stt_keywords.yaml (v4)
- 동음이의어 수정: 김→조미김/구운김, 굴→생굴/굴구이, 밤→군밤/알밤, 배→나주배
- 범용어 삭제: 여행, 관광, 축제 (오탐 유발)
- 먹방 빈출 메뉴 30+ 추가: 제육볶음, 부대찌개, 물회, 꼬막, 수육, 호떡, 빙수 등
- 관광지 명소 30+ 추가: 성산일출봉, 해운대, 감천문화마을, 남이섬, 불국사 등
- 수도권 추가: 가평, 양평, 강화도, 수원, 파주

### keyword_mapper.py
- 앞뒤 한글 경계 매칭: "거라면"→"라면" 차단, "오리지널"→"오리" 차단
- 포함관계 이중매칭 제거: "콩나물" 매칭 시 "나물" 스킵

### context_filter.py
- FOOD_AD_CATEGORIES: {"음식"}
- TRAVEL_AD_CATEGORIES: {"관광지"}

### pilot_multimodal_test.py
- CLIP 가중치 +1 → +2 상향
- 관광지 CLIP 단독 트리거 허용 (B-roll 예외)
- --save-frames: PIL 한글 렌더링, Windows 파일명 호환
- frames 빈 리스트 방어

## 4. 성능 평가

### 신호별

| 신호 | 평가 | 긴 영상 기준 |
|------|------|------------|
| STT | ⭐⭐⭐⭐ | 에이스 — 지역명/음식명 정확 |
| CLIP | ⭐⭐⭐ | 장면 분류 양호 |
| YOLO | ⭐⭐ | 보조 수준 (도메인 갭) |
| OCR | ⭐~⭐⭐ | 깨진 텍스트이나 교차검증으로 활용 가능 |

### 결론

- **트레일러 (1~3분)**: 신호 부족으로 성능 낮음 → VOD 단위 카테고리 태깅으로 대응
- **긴 영상 (20분+)**: 4종 멀티시그널 퓨전이 잘 동작 → TRIGGER 정확도 양호
- **아키텍처**: 멀티모달 교차검증 설계는 유효, 긴 콘텐츠에서 강점
