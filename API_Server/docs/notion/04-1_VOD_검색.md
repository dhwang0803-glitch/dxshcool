# GET /vod/search — GNB 통합 검색

## 기능 개요

제목, 출연진, 감독, 장르를 대상으로 VOD를 통합 검색한다. 초성 검색, 회차 검색을 지원한다.

## 기본 기능

**요청**:
```
GET /vod/search?q={검색어}
```
- `q`: 필수, 1~100자

**처리 로직**:
1. 회차 패턴 분리: `"응답하라 3화"` → `("응답하라", 3)`
2. 검색 방식 분기:
   - **초성 입력** (ㄱ~ㅎ만): `series_nm_chosung LIKE prefix%`
   - **일반 텍스트**: `search_text ILIKE` + `similarity()` 정렬 (pg_trgm)
3. 회차 지정 시: 매칭된 시리즈에서 에피소드 필터링 (`asset_nm LIKE '%{N}화%'`)

**응답**:
```json
{
  "items": [
    {
      "series_nm": "응답하라 1994",
      "asset_nm": "응답하라 1994 03회",
      "genre": "드라마",
      "ct_cl": "TV 드라마",
      "poster_url": "/posters/..."
    }
  ],
  "total": 1
}
```

### 검색 예시

| 입력 | 검색 방식 | 결과 |
|------|----------|------|
| `눈물의 여왕` | 일반 텍스트 (trgm) | 시리즈 레벨 최대 8건 |
| `ㄴㅁㅇㅇ` | 초성 검색 | 시리즈명 초성 prefix 매칭 |
| `눈물의 여왕 3` | 일반 + 회차 | 눈물의 여왕 시리즈에서 3회 에피소드 |
| `ㄴㅁ 5화` | 초성 + 회차 | 초성 매칭 시리즈에서 5회 에피소드 |

## 예외사항

- 검색 결과 없으면 `items: []`, `total: 0` (에러 아님)

## 제약사항

- 인증 불필요 (공개 검색)
- 최대 반환 건수: 8건
- MV 기반: `serving.vod_search_index` (search_text, series_nm_chosung 컬럼)
- 에피소드 중복 제거: `DISTINCT ON (asset_nm)`, SO 우선순위 kth > cjc > hcn

## 업스트림 의존성

| 테이블 | 컬럼 | 타입 | 용도 |
|--------|------|------|------|

| `public.vod` | `full_asset_id`, `series_nm`, `asset_nm`, `genre`, `ct_cl`, `poster_url` | 각종 | 회차 검색 시 에피소드 조회 |
