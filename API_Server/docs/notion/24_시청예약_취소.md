# DELETE /reservations/{reservation_id} — 시청예약 취소

## 기능 개요

시청예약을 취소(삭제)한다.

## 기본 기능

**요청**:
```
DELETE /reservations/{reservation_id}
Authorization: Bearer {token}
```

**처리 로직**:
1. `watch_reservation` 테이블에서 `reservation_id` + `user_id_fk` 조건으로 DELETE
2. 본인 예약만 삭제 가능 (user_id_fk 조건)

**응답**:
```json
{
  "deleted": true
}
```

- `deleted: false`: 해당 예약이 없거나 이미 삭제됨

## 예외사항

- 존재하지 않는 reservation_id → `deleted: false` (에러 아님)
- 다른 유저의 예약 삭제 시도 → `deleted: false`

## 제약사항

- 인증 필수
- 물리 삭제 (soft delete 아님)

## 업스트림 의존성

없음

|--------|------|------|------|
| `public.watch_reservation` | `reservation_id`, `user_id_fk` | INT/VARCHAR | DELETE |
