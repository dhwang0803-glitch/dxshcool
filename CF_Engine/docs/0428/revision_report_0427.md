## 1. CF → 협업필터링 치환 (15건)

`CF`는 Collaborative Filtering의 영문 약자. 한국어 논문 맥락에서 개념·비교·성능 지표 표현 시 `협업필터링`으로 통일.

| 위치 | 수정 전 | 수정 후 |
|------|--------|--------|
| 초록 | `CF 단독 대비 Recall@10 +18.7%` | `협업필터링 단독 대비 Recall@10 +18.7%` |
| 초록 | `협업 필터링(ALS)` | `협업필터링(ALS 행렬분해)` |
| 1.2절 RQ2 | `CF 단독 대비 하이브리드 구조(CF + 벡터 유사도 추천 + 태그 리랭킹)` | `협업필터링 단독 대비 하이브리드 구조(협업필터링 + 벡터 유사도 추천 + 태그 리랭킹)` |
| 4.3절 | `CF 후보 7개` | `협업필터링 후보 7개` |
| 5.3절 비교표 | `CF 단독` (행 레이블) | `협업필터링 단독` |
| 5.3절 비교표 | `CF 후보 7개 + 시각적 유사도 후보 3개 + 태그 리랭킹` | `협업필터링 후보 7개 + 시각적 유사도 후보 3개 + 태그 리랭킹` |
| 6.1절 성능표 | `CF 단독` (행 레이블) | `협업필터링 단독` |
| 6.1절 본문 | `CF 단독의 HR@10 0.3346` | `협업필터링 단독의 HR@10 0.3346` |
| 6.1절 본문 | `CF 단독 대비 Recall@10 +18.7%` (2회) | `협업필터링 단독 대비 Recall@10 +18.7%` |
| 6.1절 본문 | `CF 후보군에 포함되지 않는` | `협업필터링 후보군에 포함되지 않는` |
| 6.3절 세그먼트 표 | `CF HR@10` (헤더) | `협업필터링 HR@10` |
| 6.3절 본문 | `CF만으로는 시청 패턴이 불충분하지만` | `협업필터링만으로는 시청 패턴이 불충분하지만` |
| 6.3절 본문 | `CF만으로도 충분한 패턴이 확보되어` | `협업필터링만으로도 충분한 패턴이 확보되어` |
| 7절 결론 RQ2 | `CF 단독 대비 Recall@10 +18.7%` | `협업필터링 단독 대비 Recall@10 +18.7%` |

## 2. ALS → ALS 행렬분해 치환 (7건)

`ALS`는 Alternating Least Squares의 약자. 알고리즘·수식 기술 맥락에서 `ALS 행렬분해`로 확장.  
단, `iALS`는 인용 논문(Rendle et al.[9])의 고유 알고리즘명이므로 유지.

| 위치 | 수정 전 | 수정 후 |
|------|--------|--------|
| 초록 | `협업 필터링(ALS)` | `협업필터링(ALS 행렬분해)` |
| 1.2절 | `기법(ALS[6], CLIP[7], 태그 기반 리랭킹)` | `기법(ALS 행렬분해[6], CLIP[7], 태그 기반 리랭킹)` |
| 2.1절 | `전통적 기법(kNN, ALS)이` | `전통적 기법(kNN, ALS 행렬분해)이` |
| 2.3절 | `Hu et al.[6]의 ALS는 암시적 피드백` | `Hu et al.[6]의 ALS 행렬분해는 암시적 피드백` |
| 3.2절 | `ALS 잠재 벡터 학습 시` | `ALS 행렬분해 잠재 벡터 학습 시` |
| 4.1절 | `Hu et al.[6]의 ALS를 Python implicit 패키지` | `Hu et al.[6]의 ALS 행렬분해를 Python implicit 패키지` |
| 5.3절 비교표 | `ALS ($f$=128, $\alpha$=40, $\lambda$=0.01, $T$=20)` | `ALS 행렬분해 ($f$=128, $\alpha$=40, $\lambda$=0.01, $T$=20)` |
