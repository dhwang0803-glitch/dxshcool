# GET /user/me/history — 시청 내역

## 기능 개요

에피소드 시청 내역을 반환한다. `episode_progress` 기반 (watch_history raw 데이터 미사용).

## 기본 기능

**요청**:
```
GET /user/me/history?limit=50
Authorization: Bearer {token}
```
- `limit`: 1~200, 기본 50

**처리 로직**:
1. `episode_progress` 전체 (completion_rate 무관) 최신순
2. `vod` JOIN으로 메타데이터 보강

**응답**:
```json
{
  "items": [
    {
      "series_nm": "눈물의 여왕",
      "episode_title": "눈물의 여왕 05회",
      "poster_url": "/posters/...",
      "completion_rate": 73,
      "watched_at": "2026-03-19T14:30:00Z"
    }
  ],
  "total": 127
}
```

## 예외사항

- 시청 내역 없으면 빈 배열 (에러 아님)

## 제약사항

- 인증 필수
- **원칙**: `watch_history`(ML 전용)는 유저에게 직접 노출하지 않음
- `episode_progress` 데이터로 대체하여 정수 0~100% 기준 표시
- 상하 스크롤, 25건 초과 시 페이지 번호 버튼

## 업스트림 의존성

| 테이블 | 컬럼 | 타입 | 용도 |
|--------|------|------|------|
| `public.episode_progress` | `user_id_fk`, `series_nm`, `completion_rate`, `watched_at`, `vod_id_fk` | 각종 | 시청 내역 |
| `public.vod` | `full_asset_id`, `asset_nm`, `poster_url` | VARCHAR/TEXT | 메타데이터 JOIN |
