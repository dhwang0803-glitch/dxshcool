# GET /series/{series_nm}/progress — 시청 진행 현황

## 기능 개요

특정 시리즈의 에피소드별 시청 진행률을 반환한다. "이어보기" 버튼 표시에 사용.

## 기본 기능

**요청**:
```
GET /series/{series_nm}/progress
Authorization: Bearer {token}
```

**처리 로직**:
1. `episode_progress` + `vod` JOIN (user_id_fk + series_nm 기준)
2. `watched_at` DESC 정렬
3. 마지막 시청 에피소드 정보 (`last_episode`, `last_completion_rate`) 추출

**응답**:
```json
{
  "series_nm": "눈물의 여왕",
  "last_episode": "눈물의 여왕 05회",
  "last_completion_rate": 73,
  "episodes": [
    {
      "episode_title": "눈물의 여왕 05회",
      "completion_rate": 73,
      "watched_at": "2026-03-19T14:30:00Z"
    },
    {
      "episode_title": "눈물의 여왕 04회",
      "completion_rate": 100,
      "watched_at": "2026-03-18T21:00:00Z"
    }
  ]
}
```

## 예외사항

- 시청 기록 없으면 빈 배열 반환 (에러 아님)
- `last_episode` = null, `last_completion_rate` = null

## 제약사항

- 인증 필수
- `completion_rate`: SMALLINT 0~100 정수 (watch_history의 0~1 REAL과 별개)

## 업스트림 의존성

| 테이블 | 컬럼 | 타입 | 용도 |
|--------|------|------|------|
| `public.episode_progress` | `user_id_fk`, `series_nm`, `completion_rate`, `watched_at`, `vod_id_fk` | VARCHAR/SMALLINT/TIMESTAMPTZ | 진행률 데이터 |
| `public.vod` | `full_asset_id`, `asset_nm` | VARCHAR | 에피소드명 JOIN |
