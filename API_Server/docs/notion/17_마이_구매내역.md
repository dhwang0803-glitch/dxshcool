# GET /user/me/purchases — 구매 내역

## 기능 개요

포인트 구매/대여 내역을 반환한다.

## 기본 기능

**요청**:
```
GET /user/me/purchases?limit=50
Authorization: Bearer {token}
```
- `limit`: 1~200, 기본 50

**처리 로직**:
- `purchase_history` 최신순 조회

**응답**:
```json
{
  "items": [
    {
      "series_nm": "눈물의 여왕",
      "option_type": "rental",
      "points_used": 490,
      "purchased_at": "2026-03-20T15:30:00Z",
      "expires_at": "2026-03-22T15:30:00Z"
    }
  ],
  "total": 5
}
```

## 예외사항

- 구매 내역 없으면 빈 배열 (에러 아님)

## 제약사항

- 인증 필수
- legacy 백필 데이터: `points_used=0`, `option_type='permanent'` (watch_history 기반 초기 적재)
- 상하 스크롤, 25건 초과 시 페이지 번호 버튼

## 업스트림 의존성

| 테이블 | 컬럼 | 타입 | 용도 |
|--------|------|------|------|
| `public.purchase_history` | `user_id_fk`, `series_nm`, `option_type`, `points_used`, `purchased_at`, `expires_at` | 각종 | 구매 내역 |
