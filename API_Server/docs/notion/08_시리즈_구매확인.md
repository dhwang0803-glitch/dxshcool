# GET /series/{series_nm}/purchase-check — 구매 여부 확인

## 기능 개요

특정 시리즈의 구매 여부와 대여 만료 상태를 확인한다. 시리즈 상세 페이지에서 "구매하기" vs "시청하기" UI 분기에 사용.

## 기본 기능

**요청**:
```
GET /series/{series_nm}/purchase-check
Authorization: Bearer {token}
```

**처리 로직**:
1. `purchase_history`에서 해당 유저+시리즈의 최신 구매 기록 조회
2. `is_expired` 계산:
   - `permanent` -> 항상 false
   - `rental` -> `expires_at > NOW()` 체크

**응답 (구매 있음)**:
```json
{
  "series_nm": "눈물의 여왕",
  "purchased": true,
  "option_type": "rental",
  "expires_at": "2026-03-22T15:30:00Z",
  "is_expired": false
}
```

**응답 (미구매)**:
```json
{
  "series_nm": "눈물의 여왕",
  "purchased": false,
  "option_type": null,
  "expires_at": null,
  "is_expired": null
}
```

## 예외사항

- 미구매 시 에러가 아닌 `purchased: false` 반환

## 제약사항

- 인증 필수
- 대여 만료 기간: 48시간 (`purchased_at + INTERVAL '48 hours'`)
- 동일 시리즈 재구매 허용 (최신 구매 기록 기준)

## 업스트림 의존성
