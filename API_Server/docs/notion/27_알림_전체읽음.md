# POST /user/me/notifications/read-all — 전체 읽음 처리

## 기능 개요

유저의 모든 미읽음 알림을 일괄 읽음 처리한다.

## 기본 기능

**요청**:
```
POST /user/me/notifications/read-all
Authorization: Bearer {token}
```

**처리 로직**:
1. `notifications` 테이블에서 `user_id_fk` + `read = FALSE` 조건의 모든 행을 `read = TRUE`로 UPDATE

**응답**:
```json
{
  "updated": 5
}
```

- `updated`: 읽음 처리된 건수 (이미 전부 읽음이면 `0`)

## 예외사항

없음 (미읽음 없으면 `updated: 0`)

## 제약사항

- 인증 필수

## 업스트림 의존성

없음

## 다운스트림 의존성

| 테이블 | 컬럼 | 타입 | 용도 |
|--------|------|------|------|
| `public.notifications` | `user_id_fk`, `read` | VARCHAR/BOOLEAN | UPDATE read = TRUE (WHERE read = FALSE) |
