# GET /user/me/watching — 시청 중 콘텐츠

## 기능 개요

현재 시청 중인 에피소드 목록을 반환한다. 진행률 1~99% 범위 (완료/미시작 제외), 최신순.

## 기본 기능

**요청**:
```
GET /user/me/watching?limit=10
Authorization: Bearer {token}
```
- `limit`: 1~50, 기본 10

**처리 로직**:
1. `episode_progress` WHERE `completion_rate BETWEEN 1 AND 99`
2. `vod` JOIN으로 메타데이터 보강
3. `watched_at DESC` 정렬

**응답**:
```json
{
  "items": [
    {
      "series_nm": "눈물의 여왕",
      "episode_title": "눈물의 여왕 05회",
      "poster_url": "/posters/...",
      "completion_rate": 73,
      "watched_at": "2026-03-19T14:30:00Z"
    }
  ],
  "total": 3
}
```

## 예외사항

- 시청 중 콘텐츠 없으면 빈 배열 (에러 아님)

## 제약사항

- 인증 필수
- 완료(100%) 또는 미시작(0%) 제외
- `watch_history` raw 데이터 미사용 — `episode_progress` 기반

## 업스트림 의존성

| 테이블 | 컬럼 | 타입 | 용도 |
|--------|------|------|------|
| `public.episode_progress` | `user_id_fk`, `completion_rate`, `watched_at`, `vod_id_fk`, `series_nm` | 각종 | 진행률 필터 |
| `public.vod` | `full_asset_id`, `asset_nm`, `poster_url` | VARCHAR/TEXT | 메타데이터 JOIN |
