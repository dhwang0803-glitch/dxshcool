## 4. 분석 해석

### 4-1. 필터 ON vs OFF — 급락 패턴은 거의 동일

두 조건 모두 K=2 이내에서 75%, K=4 이내에서 90%의 유저가 급락하며,
분포 패턴이 사실상 동일하다.

**이유**: 급락 현상은 cold VOD 존재 여부보다
유저의 **시청 집중도(특정 시리즈에 편중)** 에 의해 결정되기 때문.
cold VOD를 추가해도 이미 1위 추천은 시청 이력 기반으로 확정되어 있어
급락 지점이 바뀌지 않는다.

### 4-2. 점수가 소폭 낮아지는 이유

필터 OFF → cold VOD(poster/embedding 없는 저품질)까지 포함한 더 넓은 풀에서 추천.
유저-아이템 행렬이 더 희소해져 ALS 잠재 벡터 품질이 소폭 저하됨.
- 급락 지점 평균 점수: 0.8622 → 0.8472 (-1.7%)
- Top-20 끝 평균 점수: 0.5011 → 0.4972 (-0.8%)

### 4-3. 1위 급락 비율 소폭 증가 (64.1% → 65.3%)

cold VOD가 포함되면 일부 유저의 2위 추천이 연관성 낮은 cold VOD로 채워져
1위와 2위 사이 점수 격차가 더 벌어짐 → 1위 급락 비율 미세 증가.

## 5. 필터 OFF가 급락에 미치는 영향 요약

| 관점 | 결론 |
|------|------|
| 급락 패턴 | **필터 ON/OFF 무관하게 동일** — 폴백 전략 설계 기준 변화 없음 |
| 추천 점수 품질 | 필터 ON이 소폭 높음 (고품질 VOD만으로 학습) |
| 1위 급락 비율 | 필터 OFF 시 1.2%p 증가 (65.3% vs 64.1%) |
| 권장 K | **K=10 유지** (필터 ON/OFF 모두 동일 결론) |

## 6. 결론

품질 필터 ON/OFF는 **급락 지점 분포에 실질적인 영향을 주지 않는다.**
Cold VOD 포함 여부와 무관하게 유저의 추천 급락은 시청 집중도에 의해 결정된다.

따라서:
- **K 설정 근거** (K=2 이내 75% 커버, K=4 이내 90% 커버)는 필터 조건과 무관하게 유효
- **폴백 전략**은 품질 필터 ON 기준(119,730개 고품질 VOD)으로 설계하는 것이 적절
- **추천 품질** 자체는 필터 ON이 더 높음 (NDCG@10 기준 +13% 개선 이미 확인)
