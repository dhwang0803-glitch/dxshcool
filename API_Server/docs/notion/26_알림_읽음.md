# PATCH /user/me/notifications/{notification_id}/read — 알림 읽음 처리

## 기능 개요

특정 알림을 읽음 처리한다.

## 기본 기능

**요청**:
```
PATCH /user/me/notifications/{notification_id}/read
Authorization: Bearer {token}
```

**처리 로직**:
1. `notifications` 테이블에서 `notification_id` + `user_id_fk` 조건으로 `read = TRUE` UPDATE

**응답**:
```json
{
  "success": true
}
```

## 예외사항

| 에러 코드 | HTTP | 조건 | 메시지 |
|-----------|------|------|--------|
| `NOTIFICATION_NOT_FOUND` | 404 | 해당 알림이 없거나 다른 유저의 알림 | 해당 알림을 찾을 수 없습니다 |

## 제약사항

- 인증 필수
- 이미 읽은 알림에 재호출해도 정상 응답 (`UPDATE 1` 성공)
- 본인 알림만 처리 가능
