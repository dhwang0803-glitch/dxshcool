# DELETE /wishlist/{series_nm} — 찜 해제

## 기능 개요

찜 목록에서 시리즈를 제거한다.

## 기본 기능

**요청**:
```
DELETE /wishlist/{series_nm}
Authorization: Bearer {token}
```

**처리 로직**:
```sql
DELETE FROM wishlist
WHERE user_id_fk = $1 AND series_nm = $2
RETURNING series_nm
```

**응답**:
```json
{
  "series_nm": "눈물의 여왕",
  "removed": true
}
```

## 예외사항

| 에러 코드 | HTTP | 조건 | 메시지 |
|-----------|------|------|--------|
| `WISHLIST_NOT_FOUND` | 404 | 찜 목록에 해당 시리즈 없음 | 찜 목록에 없는 시리즈입니다 |

## 제약사항

- 인증 필수
- DELETE 0행 (해당 찜 없음) -> 404 반환
