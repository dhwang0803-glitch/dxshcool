# POST /wishlist — 찜 추가

## 기능 개요

시리즈를 찜 목록에 추가한다. 이미 존재하면 무시 (멱등성 보장).

## 기본 기능

**요청**:
```json
POST /wishlist
Authorization: Bearer {token}
{ "series_nm": "눈물의 여왕" }
```

**처리 로직**:
```sql
INSERT INTO wishlist (user_id_fk, series_nm)
VALUES ($1, $2)
ON CONFLICT DO NOTHING
RETURNING series_nm, created_at
```

**응답**:
```json
{
  "series_nm": "눈물의 여왕",
  "created_at": "2026-03-20T15:30:00Z"
}
```

## 예외사항

- 이미 찜한 시리즈 재추가 시 **에러 아님** — 기존 데이터 반환 (ON CONFLICT DO NOTHING)

## 제약사항

- 인증 필수
- PK = `(user_id_fk, series_nm)`
- `series_nm`은 FK가 아닌 일반 텍스트 (vod 테이블의 series_nm과 논리적 매핑)
