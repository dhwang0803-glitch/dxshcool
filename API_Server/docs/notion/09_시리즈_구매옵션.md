# GET /series/{series_nm}/purchase-options — 구매 옵션

## 기능 개요

시리즈의 구매 옵션을 반환한다. FOD 무료 콘텐츠는 빈 옵션 + `is_free: true` 반환.

## 기본 기능

**요청**:
```
GET /series/{series_nm}/purchase-options
```

**처리 로직**:
1. vod 테이블에서 `series_nm`의 `asset_prod` 확인
2. `asset_prod = 'FOD'` -> 무료 (구매 불필요)
3. 유료 -> 대여/영구 2가지 옵션 반환

**응답 (FOD 무료)**:
```json
{
  "series_nm": "무료 콘텐츠",
  "is_free": true,
  "options": []
}
```

**응답 (유료)**:
```json
{
  "series_nm": "눈물의 여왕",
  "is_free": false,
  "options": [
    { "option_type": "rental", "points": 490, "duration": "48h" },
    { "option_type": "permanent", "points": 1490, "duration": null }
  ]
}
```

## 예외사항

| 에러 코드 | HTTP | 조건 | 메시지 |
|-----------|------|------|--------|
| `SERIES_NOT_FOUND` | 404 | series_nm이 vod에 없음 | 해당 시리즈를 찾을 수 없습니다 |

## 제약사항

- 인증 불필요
- FOD 시리즈: 구매 절차 없이 "1화 시청하기" UI 표시
- 유료 시리즈: "구매하기" 버튼 -> 구매 페이지 이동
- 가격: Mock 기준 확정 (대여 490P, 영구 1490P)

| `public.vod` | `series_nm`, `asset_prod` | VARCHAR | FOD 여부 확인 |
