## 핵심 인사이트

### 잘 되는 영역
- **TV드라마 / 영화**: cast·date·smry·series_nm 80~89% — TMDB 등록률 높음
- **소스 단일화**: 전 컬럼 100% TMDB — 파이프라인 단순

### 문제 영역
| 문제 유형 | 건수(추정) | 원인 |
|----------|----------:|------|
| 키즈·로컬·교육 등 TMDB 미등록 | ~22건 | TMDB에 없는 국내 제작 콘텐츠 |
| director 낮은 회수율 | 51건 실패 | TV시리즈는 감독 단독 미표기 |
| rating 미흡 | 47건 실패 | 한국 등급 미등록 콘텐츠 다수 |

### 다음 개선 포인트
1. **KMDB API 연동** — 키즈/시사교양/로컬 ~22건 보완 예상
2. **disp_rtm 보완** — TMDB runtime 단위 변환 오류 검토 (56% → 목표 80%+)
3. **director fallback** — TV시리즈 creator 필드 활용
