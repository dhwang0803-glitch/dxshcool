# DELETE /user/me/notifications/{notification_id} — 알림 삭제

## 기능 개요

특정 알림을 삭제한다.

## 기본 기능

**요청**:
```
DELETE /user/me/notifications/{notification_id}
Authorization: Bearer {token}
```

**처리 로직**:
1. `notifications` 테이블에서 `notification_id` + `user_id_fk` 조건으로 DELETE

**응답**:
```json
{
  "deleted": true
}
```

## 예외사항

| 에러 코드 | HTTP | 조건 | 메시지 |
|-----------|------|------|--------|
| `NOTIFICATION_NOT_FOUND` | 404 | 해당 알림이 없거나 다른 유저의 알림 | 해당 알림을 찾을 수 없습니다 |

## 제약사항

- 인증 필수
- 물리 삭제 (soft delete 아님)
- 본인 알림만 삭제 가능

## 업스트림 의존성

없음

| 테이블 | 컬럼 | 타입 | 용도 |
|--------|------|------|------|
| `public.notifications` | `notification_id`, `user_id_fk` | INT/VARCHAR | DELETE |
