## 핵심 인사이트

### 잘 되는 영역
- **TV드라마 / 영화**: cast·date·smry·series_nm 80~91% — TMDB 등록률 높음
- **영화**: KMDB 보완으로 cast/rating/director 모두 82% 달성

### 문제 영역

| 문제 유형 | 건수 | 원인 |
|----------|-----:|------|
| TMDB+KMDB 미등록 (전 필드 실패) | 24건 | 키즈·로컬·교육·스포츠 등 국내 비TMDB 콘텐츠 |
| TV 시리즈 rating | ~30건 | TMDB/KMDB 모두 등급 미등록 — 구조적 한계 |
| TV 연예/오락 director | ~8건 | 예능 감독 개념 부재, TMDB/KMDB 미표기 |

### 다음 개선 포인트
1. **disp_rtm 보완** — TMDB runtime 단위 변환 로직 재검토 (56% → 목표 80%+)
2. **키즈/로컬 콘텐츠** — 추가 데이터 소스 검토 필요 (현재 TMDB+KMDB로도 커버 불가)
3. **TV 시리즈 rating** — 구조적 한계, 수동 입력 또는 방송통신심의위원회 API 검토
