# GET /recommend/{user_id} — 개인화 추천

> 최종 수정: 2026-03-31

## 기능 개요

hybrid_recommendation(top_vod 10건) + tag_recommendation(패턴 그룹핑) + 벡터 유사도 + rec_sentence 기반 개인화 추천을 반환한다. 추천 데이터 없으면 popular_recommendation fallback.

## 기본 기능

**요청**:
```
GET /recommend/{user_id}
Authorization: Bearer {token}
```

**처리 로직**:
1. **top_vod** (배열 10건): `serving.hybrid_recommendation`에서 score 내림차순 → backdrop_url IS NOT NULL인 VOD만 선택. 부족분은 `cold_genre_detail`(연령대 기반)로 보충. 각 VOD에 `rec_sentence`(세그먼트별 LLM 추천 문구) 첨부
2. **patterns**: `serving.tag_recommendation`에서 태그별 그룹핑
   - 태그 카테고리: `genre_detail`, `director`, `actor_lead`, `actor_guest`, `cold_genre_detail`
   - 배우 태그(`actor_guest`/`director`) + TV 연예/오락: 에피소드 단위 유지 (게스트 출연 반영)
   - 그 외: series_nm 기준 중복 제거
   - 그룹핑 키: `(tag_category, tag_value)`
3. **벡터 유사도**: `user_embedding` meta part(384D) ↔ `vod_series_embedding` cosine 유사도 top 10
4. 모두 없으면 `serving.popular_recommendation`에서 인기순 fallback

**응답**:
```json
{
  "user_id": "a1b2c3...",
  "top_vod": [
    {
      "vod_id": "cjc|M1234...",
      "series_id": "기생충",
      "asset_nm": "기생충",
      "poster_url": "https://...",
      "backdrop_url": "https://...",
      "rec_sentence": "빈부격차라는 사회적 주제를 블랙코미디로 풀어낸 봉준호 감독의 걸작"
    }
  ],
  "patterns": [
    {
      "pattern_rank": 1,
      "pattern_reason": "봉준호 감독 작품을 즐겨 보셨어요",
      "vod_list": [
        {
          "series_id": "옥자",
          "asset_nm": "옥자",
          "poster_url": "https://...",
          "score": 0.87
        }
      ]
    }
  ],
  "source": "personalized"
}
```

### 필드 상세

| 필드 | 타입 | 설명 |
|------|------|------|
| `top_vod` | `array` | 개인화 추천 top 10 VOD 배열 (히어로 캐러셀) |
| `top_vod[].vod_id` | `string` | full_asset_id (PK) |
| `top_vod[].series_id` | `string` | series_nm 기준 시리즈명 |
| `top_vod[].asset_nm` | `string` | 자산명 |
| `top_vod[].poster_url` | `string` | 포스터 URL |
| `top_vod[].backdrop_url` | `string` | 가로형 배경 이미지 URL (IS NOT NULL 필터) |
| `top_vod[].rec_sentence` | `string \| null` | LLM 생성 추천 문구 (세그먼트별 개인화) |
| `patterns[].pattern_rank` | `int` | 패턴 순서 (1부터) |
| `patterns[].pattern_reason` | `string` | 패턴 설명 라벨 |
| `patterns[].vod_list[].series_id` | `string` | series_nm 기준 시리즈명 |
| `patterns[].vod_list[].asset_nm` | `string` | 자산명 |
| `patterns[].vod_list[].poster_url` | `string \| null` | 포스터 URL |
| `patterns[].vod_list[].score` | `float \| null` | 추천 점수 |
| `source` | `string` | `"personalized"` 또는 `"popular_fallback"` |

### pattern_reason 템플릿

| tag_category | 생성 문구 |
|-------------|----------|
| `genre_detail` | {value} 장르를 즐겨 보셨어요 |
| `director` | {value} 감독 작품을 즐겨 보셨어요 |
| `actor_lead` | {value} 배우 출연작을 자주 보셨어요 |
| `actor_guest` | {value} 배우가 출연한 프로그램을 모아봤어요 |
| `cold_genre_detail` | {유저앞5자}님이 좋아할만한 {value} 시리즈 |
| 벡터 유사도 | 나의 취향과 비슷한 콘텐츠 (고정) |

### 패턴 순서

```
1~N: 개인화 태그 패턴 (genre_detail → director → actor_lead → actor_guest)
N+1~: Cold Start 패턴 (cold_genre_detail) — 시청 이력 적은 유저만
마지막: 벡터 유사도 패턴 ("나의 취향과 비슷한 콘텐츠")
```

### Fallback 동작

| 조건 | `source` | 내용 |
|------|----------|------|
| 개인화 데이터 있음 | `"personalized"` | top_vod 10건 + 태그/cold/벡터 패턴 |
| 개인화 데이터 없음 | `"popular_fallback"` | top_vod = 인기 top 5, patterns = 인기 6~10위 |

### 테스터 격리

`public."user".is_test = TRUE`인 유저는 자동으로 `*_test` 격리 테이블(`hybrid_recommendation_test`, `tag_recommendation_test`) 경유.

## 예외사항

- 추천/인기 모두 없으면 `top_vod: []`, `patterns: []` (에러 아님)
- `serving` 스키마 미생성 시 자동 fallback (Exception 무시)

## 제약사항

- 인증 필수
- TTL 필터: `expires_at IS NULL OR expires_at > NOW()` (만료된 추천 제외)
- backdrop_url IS NOT NULL인 VOD만 top_vod에 포함

## 업스트림 의존성

| 테이블 | 컬럼 | 용도 |
|--------|------|------|
| `serving.hybrid_recommendation` | `user_id_fk`, `vod_id_fk`, `score`, `expires_at` | top_vod score 내림차순 10건 |
| `serving.tag_recommendation` | `user_id_fk`, `tag_category`, `tag_value`, `tag_rank`, `tag_affinity`, `vod_id_fk`, `vod_rank`, `vod_score`, `expires_at` | patterns 태그별 그룹핑 |
| `serving.popular_recommendation` | `vod_id_fk`, `score` | 인기순 fallback |
| `serving.rec_sentence` | `vod_id_fk`, `segment_id`, `rec_sentence` | top_vod 추천 문구 (시리즈 기반 전파) |
| `public.user_segment` | `user_id_fk`, `segment_id` | 유저 세그먼트 조회 (rec_sentence 매칭) |
| `public.user_embedding` | `user_id_fk`, `embedding` | 벡터 유사도: meta part [513:896] 384D |
| `public.vod_series_embedding` | `series_nm`, `embedding`, `poster_url` | 벡터 유사도: cosine distance 상위 10 |
| `public.vod` | `series_nm`, `asset_nm`, `poster_url`, `backdrop_url`, `ct_cl` | 메타데이터 JOIN |
