# GET /series/{series_nm}/episodes — 에피소드 목록

## 기능 개요

시리즈의 에피소드 목록을 반환한다. 동일 에피소드가 다른 provider에서 중복 제공되는 경우 우선순위에 따라 1개만 반환.

## 기본 기능

**요청**:
```
GET /series/{series_nm}/episodes
```
- `series_nm`: URL 인코딩된 시리즈명 (예: `눈물의%20여왕`)

**처리 로직**:
1. `series_nm` 기준 vod 테이블 조회
2. `DISTINCT ON(asset_nm)` + provider 우선순위로 중복 제거
   - 우선순위: `kth(1) > cjc(2) > hcn(3) > 기타(4)`
   - `SPLIT_PART(full_asset_id, '|', 1)` 로 provider 추출
3. 각 에피소드에 `is_free` 포함

**응답**:
```json
{
  "series_nm": "눈물의 여왕",
  "episodes": [
    {
      "episode_title": "눈물의 여왕 01회",
      "category": "TV드라마",
      "poster_url": "/posters/...",
      "is_free": false
    }
  ],
  "total": 16
}
```

## 예외사항

| 에러 코드 | HTTP | 조건 | 메시지 |
|-----------|------|------|--------|
| `SERIES_NOT_FOUND` | 404 | series_nm에 해당하는 에피소드 없음 | 해당 시리즈를 찾을 수 없습니다 |

## 제약사항

- 인증 불필요
- 동일 에피소드 중복: `짝 08회` -> `cjc|M4903632LSGH50604201`, `cjc|M4903632LSGL21170301` (동일 provider, asset_id 상이) -> `DISTINCT ON(asset_nm)`으로 1개만
- 다회차 시리즈 (아는형님, 무한도전 등 수백 회차): **25건/페이지** 페이지네이션 필수
- 에피소드 검색: 출연진 초성검색 + 회차 숫자 검색 (구현 예정)
- 커버링 인덱스 `idx_vod_series_nm_cover` 활용

## 업스트림 의존성

| 테이블 | 컬럼 | 타입 | 용도 |
|--------|------|------|------|
| `public.vod` | `series_nm` | VARCHAR(255) | 시리즈 필터 |
| `public.vod` | `full_asset_id`, `asset_nm`, `ct_cl`, `poster_url`, `asset_prod` | 각종 | 에피소드 정보 + provider 추출 + FOD 판단 |
