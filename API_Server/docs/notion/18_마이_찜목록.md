# GET /user/me/wishlist — 찜 목록

## 기능 개요

유저가 찜한 시리즈 목록을 반환한다. 포스터 URL 포함, 최신순 정렬.

## 기본 기능

**요청**:
```
GET /user/me/wishlist
Authorization: Bearer {token}
```

**처리 로직**:
1. `wishlist` 테이블에서 유저의 찜 목록 조회
2. `vod` 서브쿼리로 `poster_url` 보강:
   ```sql
   SELECT w.series_nm, w.created_at,
          (SELECT v.poster_url FROM vod v WHERE v.series_nm = w.series_nm LIMIT 1) AS poster_url
   FROM wishlist w WHERE w.user_id_fk = $1
   ORDER BY w.created_at DESC
   ```

**응답**:
```json
{
  "items": [
    {
      "series_nm": "눈물의 여왕",
      "poster_url": "/posters/...",
      "created_at": "2026-03-19T10:00:00Z"
    }
  ],
  "total": 2
}
```

## 예외사항

- 찜 목록 비어있으면 빈 배열 (에러 아님)

## 제약사항

- 인증 필수
- poster_url은 상관 서브쿼리로 조회 (series_nm 기준, LIMIT 1)
- series_nm에 해당하는 VOD가 없으면 poster_url = null

## 업스트림 의존성

| 테이블 | 컬럼 | 타입 | 용도 |
|--------|------|------|------|
| `public.wishlist` | `user_id_fk`, `series_nm`, `created_at` | VARCHAR/TIMESTAMPTZ | 찜 목록 |
| `public.vod` | `series_nm`, `poster_url` | VARCHAR/TEXT | 포스터 서브쿼리 |
