# GET /home/sections — CT_CL별 인기 섹션

## 기능 개요

CT_CL 4종(영화, TV드라마, TV 연예/오락, TV애니메이션)별 인기 top 20을 반환한다.
홈 화면의 장르별 캐러셀 섹션 데이터.

## 기본 기능

**요청**:
```
GET /home/sections
```

**처리 로직**:
1. `serving.popular_recommendation` 테이블에서 ct_cl별 그룹핑
2. 각 ct_cl당 rank 순서로 20개 조회
3. vod 테이블 JOIN으로 메타데이터 보강

**응답**:
```json
{
  "sections": [
    {
      "ct_cl": "영화",
      "vod_list": [
        {
          "series_nm": "범죄도시4",
          "title": "범죄도시4",
          "poster_url": "/posters/...",
          "score": 0.92,
          "rank": 1
        }
      ]
    }
  ]
}
```

## 예외사항

- 특정 ct_cl 데이터 없으면 해당 섹션 빈 배열 (에러 아님)

## 제약사항

- 인증 불필요
- CT_CL 4종: 영화, TV드라마, TV 연예/오락, TV애니메이션
- 좌우 스크롤 구조, 25건 초과 시 페이지 번호 버튼

## 업스트림 의존성

| 테이블 | 컬럼 | 타입 | 용도 |
|--------|------|------|------|
| `serving.popular_recommendation` | `ct_cl`, `rank`, `vod_id_fk`, `score`, `recommendation_type` | VARCHAR(64)/SMALLINT/REAL/VARCHAR(32) | CT_CL별 Top-N |
| `public.vod` | `series_nm`, `asset_nm`, `poster_url` | VARCHAR/TEXT | 메타데이터 JOIN |
