# GET /recommend/{user_id} — 개인화 추천

## 기능 개요

hybrid_recommendation(top_vod) + tag_recommendation(패턴 그룹핑) 기반 개인화 추천을 반환한다. 추천 데이터 없으면 popular_recommendation fallback.

## 기본 기능

**요청**:
```
GET /recommend/{user_id}
Authorization: Bearer {token}
```

**처리 로직**:
1. **top_vod**: `serving.hybrid_recommendation`에서 rank 순 조회 → poster_url 있는 최상위 VOD 우선 선택
2. **patterns**: `serving.tag_recommendation`에서 태그별 그룹핑 (top 5 태그 × top 10 VOD)
   - 태그 카테고리별 pattern_reason 자동 생성 (감독/배우/장르/등급)
   - 배우 태그 + TV 연예/오락: 에피소드 단위 유지 (게스트 출연 반영)
   - 그 외: series_nm 기준 중복 제거
3. 둘 다 없으면 `serving.popular_recommendation`에서 인기순 fallback

**응답**:
```json
{
  "user_id": "a1b2c3...",
  "top_vod": {
    "series_id": "cjc|M1234...",
    "asset_nm": "기생충",
    "poster_url": "/posters/..."
  },
  "patterns": [
    {
      "pattern_rank": 1,
      "pattern_reason": "봉준호 감독 작품을 즐겨 보셨어요",
      "vod_list": [
        {
          "series_id": "cjc|M5678...",
          "asset_nm": "옥자",
          "poster_url": "/posters/...",
          "score": 0.87
        }
      ]
    }
  ],
  "source": "personalized"
}
```

- `source`: `"personalized"` (추천 성공) | `"popular_fallback"` (인기 대체)
- `top_vod`: 최상위 추천 VOD 1건 (poster_url 있는 항목 우선). null 가능
- `patterns`: 태그별 그룹핑 배열. fallback 시 "지금 인기 있는 콘텐츠" 1개 패턴

### pattern_reason 템플릿

| tag_category | 생성 문구 |
|-------------|----------|
| `director` | {value} 감독 작품을 즐겨 보셨어요 |
| `actor` | {value} 배우 출연작을 자주 보셨어요 |
| `genre` | {value} 장르를 자주 시청하셨네요 |
| `genre_detail` | {value} 장르를 즐겨 보시네요 |
| `rating` | {value} 등급 콘텐츠를 선호하시네요 |

## 예외사항

- 추천/인기 모두 없으면 `top_vod: null`, `patterns: []` (에러 아님)
- `serving` 스키마 미생성 시 자동 fallback (Exception 무시)

## 제약사항

- 인증 필수
- TTL 필터: `expires_at IS NULL OR expires_at > NOW()` (만료된 추천 제외)

## 업스트림 의존성

| 테이블 | 컬럼 | 타입 | 용도 |
|--------|------|------|------|
| `serving.hybrid_recommendation` | `user_id_fk`, `vod_id_fk`, `rank`, `expires_at` | 각종 | top_vod 선택 (poster_url 있는 최상위) |
| `serving.tag_recommendation` | `user_id_fk`, `tag_category`, `tag_value`, `tag_rank`, `tag_affinity`, `vod_id_fk`, `vod_rank`, `vod_score`, `expires_at` | 각종 | patterns 태그별 그룹핑 |
| `serving.popular_recommendation` | `vod_id_fk`, `score`, `ct_cl` | 각종 | 인기순 fallback |
| `public.vod` | `series_nm`, `asset_nm`, `poster_url`, `ct_cl` | VARCHAR/TEXT | 메타데이터 JOIN |
