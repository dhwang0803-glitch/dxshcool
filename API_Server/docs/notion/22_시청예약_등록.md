# POST /reservations — 시청예약 등록

## 기능 개요

시청예약을 등록한다. `alert_at` 도래 시 background task가 WebSocket으로 알림을 push한다.

## 기본 기능

**요청**:
```json
POST /reservations
Authorization: Bearer {token}
{
  "channel": 25,
  "program_name": "제철장터",
  "alert_at": "2026-03-25T14:00:00Z"
}
```

| 필드 | 타입 | 필수 | 설명 |
|------|------|------|------|
| `channel` | int | O | 채널 번호 |
| `program_name` | string | O | 프로그램명 |
| `alert_at` | datetime | O | 알림 시각 (UTC) |

**처리 로직**:
1. `watch_reservation` 테이블에 INSERT
2. 생성된 예약 정보 RETURNING

**응답**:
```json
{
  "reservation_id": 42,
  "channel": 25,
  "program_name": "제철장터",
  "alert_at": "2026-03-25T14:00:00Z"
}
```

## 예외사항

없음 (INSERT 실패 시 500)

## 제약사항

- 인증 필수
- 광고 팝업(`/ad/popup`)의 `reserve_watch` 액션과 연동 예정
