# GET /user/me/points — 포인트 잔액 + 내역

## 기능 개요

포인트 잔액과 최근 적립/사용 내역을 반환한다.

## 기본 기능

**요청**:
```
GET /user/me/points?limit=20
Authorization: Bearer {token}
```
- `limit`: 1~100, 기본 20

**처리 로직**:
1. 잔액 집계: `SUM(CASE WHEN type='earn' THEN amount ELSE -amount END)`
2. 최근 내역: `created_at DESC` 정렬

**응답**:
```json
{
  "balance": 99510,
  "history": [
    {
      "type": "use",
      "amount": 490,
      "description": "눈물의 여왕 대여 구매",
      "created_at": "2026-03-20T15:30:00Z"
    },
    {
      "type": "earn",
      "amount": 100000,
      "description": "초기 포인트 지급",
      "created_at": "2026-03-01T00:00:00Z"
    }
  ]
}
```

## 예외사항

- 포인트 내역 없으면 `balance: 0`, `history: []` (에러 아님)

## 제약사항

- 인증 필수
- `type`: `earn`(적립) / `use`(사용) 2종
- 잔액은 매 요청 시 실시간 집계

## 업스트림 의존성

| 테이블 | 컬럼 | 타입 | 용도 |
|--------|------|------|------|
| `public.point_history` | `user_id_fk`, `type`, `amount`, `description`, `created_at` | 각종 | 잔액 집계 + 내역 |
