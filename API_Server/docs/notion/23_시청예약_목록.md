# GET /reservations — 시청예약 목록

## 기능 개요

현재 유저의 미알림 시청예약 목록을 조회한다.

## 기본 기능

**요청**:
```
GET /reservations
Authorization: Bearer {token}
```

**처리 로직**:
1. `watch_reservation` 테이블에서 `user_id_fk` + `notified = FALSE` 조건 조회
2. `alert_at ASC` 정렬 (가장 빨리 도래하는 예약 먼저)

**응답**:
```json
{
  "items": [
    {
      "reservation_id": 42,
      "channel": 25,
      "program_name": "제철장터",
      "alert_at": "2026-03-25T14:00:00Z"
    }
  ],
  "total": 1
}
```

## 예외사항

- 예약 없으면 `items: []`, `total: 0` (에러 아님)

## 제약사항

- 인증 필수
- 알림 전송 완료(`notified = TRUE`) 항목은 조회 대상에서 제외

## 업스트림 의존성

| 테이블 | 컬럼 | 타입 | 용도 |
|--------|------|------|------|
| `public.watch_reservation` | `reservation_id`, `channel`, `program_name`, `alert_at`, `notified` | 각종 | 미알림 예약 조회 |
